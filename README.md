#  Todo Backend
 <h5> To start the server</h5>
 <br/>
 1. cd todo-api
  <br/>
 2. npm install
  <br/>
 3. npm run dev
  <br/>
 - npm run dev
 <h5>Packages used </h5>
- sqlite3
- express
- cors
- body-parser
<h2> Api Urls </h2>
Base URL ->
http://localhost:3000
</br>
1. Create a TODO
Endpoint: POST /todos
</br>
2. Get All TODOs
Endpoint: GET /todos
</br>
3. Update a TODO
Endpoint: PUT /todos/:id  
</br>
4. Delete a TODO
Endpoint: DELETE /todos/:id
